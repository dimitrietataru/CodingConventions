### ✔ Use PascalCasing when abbreviating three or more characters ✔
###

> When there are two characters, both are uppercase.

### ✔
``` csharp
HtmlHelper htmlHelper;
```
``` csharp
XmlDocument xmlDocument
```
``` csharp
IOException ioException;
```
``` csharp
UIElement uiElement;
```

### ❌
``` csharp
HTMLHelper htmlHelper;
```
``` csharp
XMLDocument xmlDocument
```
``` csharp
IoException ioException;
```
``` csharp
UiElement uiElement;
```
