### ❌ Do not leave blank lines at start or end of a file ❌

> Files should not start with one, or more blank lines.  
> Files should not end with multiple blank lines.

### ✔
``` csharp
using System

namespace Program
{
    public class Application
    {
        // ...
    }
}
```

### ❌ 
``` csharp


using System

namespace Program
{
    public class Application
    {
        // ...
    }
}


```
