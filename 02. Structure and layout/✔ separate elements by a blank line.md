# **DO** separate elements by a blank line

> Add a blank line between adjacent elements.

``` csharp
// DO
public Constructor()
{
    // ...
}

public void Method1()
{
    // ...
}

public void Method2()
{
    // ...
}
```

``` csharp
// DON'T
public Constructor()
{
    // ...
}
public void Method1()
{
    // ...
}
public void Method2()
{
    // ...
}
```
