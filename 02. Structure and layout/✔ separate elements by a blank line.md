### ✔ Separate elements by a blank line ✔

> Add a blank line between adjacent elements.

### ✔
``` csharp
public Constructor()
{
    // ...
}

public void Method1()
{
    // ...
}

public void Method2()
{
    // ...
}
```

### ❌ 
``` csharp
public Constructor()
{
    // ...
}
public void Method1()
{
    // ...
}
public void Method2()
{
    // ...
}
```
